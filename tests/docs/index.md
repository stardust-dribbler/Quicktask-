# QuickTask Documentation

QuickTask is a simple command-line to-do app written in Python.

## Commands

| Command | Description | Example |
|----------|--------------|----------|
| `add` | Add a new task | `python src/main.py add "Buy groceries"` |
| `list` | List all tasks | `python src/main.py list` |
| `done` | Mark a task done | `python src/main.py done 1` |

## File Storage
Tasks are stored in `tasks.json` in the root folder.
